# Code for: Risk of permanent environmental change drives cooperation in repeated prisoner's dilemma

This repository contains the code used to generate the results and figures in the paper:

**"Risk of permanent environmental change drives cooperation in repeated prisoner's dilemma"**
Authors: Abir Utthasani, Christian Hilbe

## Overview

The repository contains the main scripts used to generate the figures in the paper.
The Jupyter notebooks reproduce the simulations and generate the plots corresponding to each figure.

## Data

The generated datasets used in the paper are stored in a separate repository:

[link-to-data-repository]


## Reproducing figures

Each notebook in the `notebooks/` directory reproduces the data analysis and figures corresponding to the main text of the paper.

## Requirements

Python ≥ 3.x with required packages: 
numpy
matplotlib
Jupyter Notebook

## Additional code (Mathematica)

The repository also contains scripts written in Mathematica (.nb / .wl files) used for symbolic calculations and analytical derivations related to the paper. 
These files require Wolfram Mathematica to run.
